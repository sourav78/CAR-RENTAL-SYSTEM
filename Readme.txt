About car rental system :-
-----------------------

-> Car Rental App is a console application that simulates a car rental booking system by allowing 
   the user to book a car of their choice for a period of time.

How it's works :-
--------------

-> The user first login/register himself
-> The user select a car
-> The user enter the number of days whant to rent the car and fill some informatin
-> When the user click the confirm button its automatically calculate the cost of car rent and 
   also calculate after discount the cost of car rent.
